Main file -> main.m

'noisy_audios/' contains noisy signals including the provided signal.
'useful_audios.' contains the output audio after clustering the useful part
'annotated_audios/' contains the manually clustered signals as a binary vector in .mat format with the same name as the one in the 'noisy_signals' folder.

The 'example_parameter_search.txt' file contains Jaccard similarity indexes for an example parameter space.

The rest .m files are all functions
